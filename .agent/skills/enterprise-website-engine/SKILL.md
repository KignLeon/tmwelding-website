    *   Left: Messaging, CTAs, trust signals

    *   Image supports the message

#### üì± Mobile (sm and below) ‚Äî NON-NEGOTIABLE

On mobile, the hero image becomes the primary entry point.

**Required Mobile Hero Order:**
1.  **Hero Image (FIRST)**
    *   Front-and-center
    *   Visible immediately on load
    *   Represents the business, mission, or outcome
2.  **Hero Headline & Subheadline**
3.  **Hero Badge / Trust Signals / CTA**

**Rationale:**
*   Mobile users scan visually first
*   Immediate context builds trust faster than copy
*   Image establishes legitimacy before persuasion
*   Aligns with modern conversion standards (Apple, Airbnb, Shopify)

#### üß± Technical Enforcement Rules
*   Use one hero markup only
*   Control layout strictly via responsive CSS utilities
*   No duplicated components
*   No JavaScript layout logic
*   No mobile-only hero variants

#### üö´ Anti-Patterns (FORBIDDEN)
*   Text-first mobile hero
*   Image hidden below the fold
*   Side-aligned mobile images
*   Desktop-first stacking without mobile intent
*   Hero images treated as background-only on mobile

#### ‚úÖ Conversion Standard

A hero passes QA only if:
*   On mobile, the image is the first thing the user sees
*   On desktop/tablet, messaging remains primary
*   No layout regression across breakpoints

## Review & Build Checklist

Before final output, the agent must verify:

**Design & UX**
*   Does the hero immediately communicate value?
*   Is the primary CTA visible without scrolling?
*   Is mobile behavior intentional and optimized?
*   Are trust signals visible early?

**Conversion**
*   Are forms frictionless and validated?
*   Are CTAs clear, calm, and outcome-driven?
*   Does the page guide attention logically?

**Copy**
*   Is every sentence benefit-driven?
*   Does it sound custom-built for the niche?
*   Is there zero filler or generic phrasing?

**Technical**
*   Does it deploy cleanly on GitHub Pages?
*   Are URLs clean and functional?
*   Are all assets loading correctly?
*   Is performance optimized?

**SEO & Accessibility**
*   Correct heading structure?
*   Alt text and aria labels present?
*   Metadata complete and accurate?
